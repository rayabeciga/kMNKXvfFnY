# 项目简介

## 核心功能
- 基于 YOLOv8 的石头剪刀布手势检测系统
- 支持 8500 张标注数据集，包含 '布'、'石头'、'剪刀' 三类
- 提供完整源码、训练好的模型及训练结果
- 支持图片、视频及摄像头实时检测
- 界面显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有结果
- 可自定义置信度和 IOU 阈值
- 支持选择显示标签和原图
- 提供多种检测模型选择
- 包含完整的项目运行教程

## 适用场景
- 手势识别研究与应用
- 计算机视觉教学演示
- 交互式系统开发
- 实时检测系统原型构建
- 批量图像分析处理

## 技术特点
- 基于 Python 3.9 开发
- 使用 PyCharm/VSCode + Anaconda 环境
- 集成 OpenCV-Python、PyQt5、Ultralytics、Torch1.9 等库
- 提供图形用户界面，操作简便
- 模块化设计，便于扩展和修改
